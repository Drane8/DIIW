Definición de estilos

Introducción

Ir arriba

Las hojas de estilo están compuestas por una o más reglas aplicadas a un documento HTML. Cada regla está compuesta de dos partes: Selector{declaracion; propiedades:valor;}

  • Selector: puede ser una etiqueta HTML, una clase definida por el usuario, un nombre de identificador o un modo compuesto.
  • Declaración: establece las propiedades y valores que definen el estilo.

Aplicación de estilos en selectores

Ir arriba

Referencia a una clase

Esta clase de estilo sólo se aplicaría al tipo de etiqueta o directiva, con un nombre de clase determinado: Selector.nombreclase{propiedad:valor; propiedad:valor;...}

En el siguiente ejemplo las clases resaltado y normal, únicamente se aplicarían a párrafos:

p.resaltado{background:blue} p.normal{background:yellow}

Definición de una clase

Podemos definir una clase sin hacer referencia a ningún selector en particular, esto permite utilizar la clase en cualquier etiqueta que admita la aplicación de estilo utilizando el atributo class.

.resaltado{background-color:blue}

Definición de un estilo con identificador

También podemos crear un estilo dedicado a un identificador (recomendable que no empiece por un número):

#ident01{background-color:black}

Selector universal

Podemos crear un selec tor que se aplica a todas las etiquetas, mediante el asterisco, *. En el siguiente ejemplo se dice que ponga el fondo de todas las etiquetas de color negro:

*{background-color: black}

Selectior limitado

Podemos crear un estilo que se aplicaría a una etiqueta, siempre y cuando dependa de otras:

tr td ol{background: black}

Esto significa que el estilo se aplica a etiquetas ol que están dentro de un bloque td que por su parte están dentro de un bloque tr. La dependencia es estricta, sin etiquetas intermedias diferentes a las citadas, por ejemplo una th. Atención, no confundir la definición anterior con esta, que significa que el estilo se aplica a las tres etiquetas:

Tr, td, ol{background: black}

También es posible codificar el selector universal * para incluir diversas etiquetas intermedias.

tr * ol{background: black}

Limitación a un atributo de un selector

Podemos acotar el alcance de un estilo a un elemento que tenga un atributo determinado con un valor específico. Esta es la sintaxis: elemento[atributo="valor" {propiedad:valor;..}

Por ejemplo si se quiere definir un estilo para los elementos table que tengan definido un atributo border=2:

table[border="2"{width:"100%"

Tambien se puede combinar:

table.cl03[border="2"[color="green"{width:"100%"

Tipos de relaciones

El elemento HTML es el nodo raíz del cual depende toda la estructura. En la estructura se pueden diferenciar estos tipos de relaciones:

  • Ancestros y descendientes.
  • Padres e hijos.
  • Hermanos.

Selección por jerarquía

  • Para especificar la relación entre ancestros y descendientes los elementos se separan por un espacio, puede haber etiquetas intermedias.
  • Lo mismo pero expresado en relación padre-hijo.
  • Para expresar la relación de hermanos se especifican los elementos separados por el signo +. Este código va dirigido a un elemento p que tenga un hermano table.

También es posible realizar combinaciones; el siguiente estilo va dirigido sólo a los elementos table y p que sean hermanos e hijos del padre div.

div > table + p {}
div.c2  li {}
div.c2 > li {}
table + p {}
